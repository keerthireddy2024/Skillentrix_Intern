# 🌐 S Keerthi Reddy - Portfolio Website

A modern, responsive portfolio website built using HTML, CSS, and JavaScript.  
This project showcases my frontend development skills along with real-world features like form handling, local storage, and UI/UX design.

---

## 🚀 Features

- ✅ Responsive design (mobile-friendly)
- 🌙 Light/Dark mode with localStorage
- 🔗 Smooth scrolling navigation
- 📩 Contact form with validation
- 💾 Data stored using browser localStorage
- 🔐 Admin panel to view submitted messages
- 🗑 Delete messages functionality
- 🎨 Clean and modern UI design

---

## 🛠️ Technologies Used

- HTML5
- CSS3 (Flexbox, Grid, Media Queries)
- JavaScript (ES6+)
- LocalStorage API
